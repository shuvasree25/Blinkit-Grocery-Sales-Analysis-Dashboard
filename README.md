# Blinkit-Grocery-Sales-Analysis-Dashboard
Project Title :                                                                                                                                              Blinkit Grocery Sales Analysis Dashboard the goal of this dashboard is to analyze sales performance, outlet efficiency, product distribution, and customer ratings to support data-driven business decisions. 
Purpose:                                                                                                                                                     The Blinkit Grocery Sales  Analysis Dashboard analyzes total and average sales, number of items sold, average customer ratings, outlet establishment, product fat content(low fat, regular), items type, outlet size, outlet location, and outlet type. It helps in understanding sales trends, product distribution, and outlet performance to improve business. This dashboard focuses on highlighting customer ratings , distribution across locations and outlet types.
